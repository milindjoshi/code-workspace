# C++ Task Manager (File-based)

Simple command-line task manager that stores data in a text file.

## Features

- Add tasks with priority levels
- View all tasks
- Mark tasks as complete
- Delete tasks
- Persistent storage in text file

## Compilation
```bash
make
# or
g++ -Wall -Wextra -std=c++11 -o task_manager task_manager.cpp
```

## Usage
```bash
./task_manager
```

## Technologies

- C++11
- File I/O
- STL (vector, string)
